# vimrc

## Prerequisite

```bash
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

## Usage

```
Download `.vimrc` into `~/.vimrc`
```

Install all plugins, run command in terminal and wait to install all plugin:

```bash
vim -c 'PluginInstal' -c 'qa'
```
